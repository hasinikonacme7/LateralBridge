# LateralBridge - HTML/CSS/JavaScript Website

Welcome to **LateralBridge** - A comprehensive educational platform for lateral entry B.Tech students, built with pure HTML, CSS, and JavaScript.

## Features

- **Rich Dark Theme** - Elegant purple and teal color scheme
- **Fully Responsive** - Works perfectly on mobile, tablet, and desktop
- **7 Complete Pages** - All sections fully functional
- **Interactive Elements** - Quizzes, forms, expandable modules
- **Photography Integration** - High-quality student imagery throughout
- **GeeksforGeeks Integration** - Direct links to learning resources

## File Structure

```
public
├── index.html              # Home page
├── about.html              # About lateral entry challenges
├── learning-bridge.html    # Foundation courses with GeeksforGeeks links
├── self-assessment.html    # Interactive quiz system
├── mentor-support.html     # Mentor profiles and study groups
├── transition-toolkit.html # Study resources and materials
├── feedback.html           # Contact form and FAQ
├── styles.css              # Main stylesheet
├── script.js               # JavaScript functionality
└── README.md               # This file
```

## How to Open the Website

### Option 1: Direct File Opening
1. Navigate to the `/public/` folder
2. Double-click `index.html`
3. Your default browser will open the website

### Option 2: Using Live Server (Recommended)
1. Install a live server extension in your code editor
2. Right-click on `index.html`
3. Select "Open with Live Server"
